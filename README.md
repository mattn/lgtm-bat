# lgtm-bat

Windows port of https://github.com/motemen/lgtm.sh

[![LGTM](http://lgtm.herokuapp.com/http://38.media.tumblr.com/f1042f48459acd3b1e7c568c0faa7eec/tumblr_n8o30zwsJx1tfp3xbo1_500.gif)](http://lgtm.herokuapp.com/)
